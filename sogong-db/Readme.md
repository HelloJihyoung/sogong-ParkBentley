
### npm에 mysql, ejs 확장자 설치
```
npm install mysql
npm install ejs --save
```

### mysql에 사용할 DB와 TABLE 생성
```
CREATE DATABASE my_db;

(create_db_table.txt 참고)
```

### npm 실행 
```
(npm start) or (node app.js) 
```
